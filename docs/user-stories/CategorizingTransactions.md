A user should be able to go through their transactions and categorize them individually. They should be able to split a transaction into a debit from one account and credits from multiple accounts.

Transactions should be sorted newest to oldest by default

They should be able to filter transactions by category (including uncategorized), account, type (deposit/withdrawal), date, and just search for descriptions of transactions.

In the transactions view they should be able to add journal entries

Some transactions will be transfers between accounts. A user should be able to mark a transaction as a transfer, and then be prompted with the matching transaction in the other account.
