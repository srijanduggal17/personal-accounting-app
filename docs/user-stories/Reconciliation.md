A user should be able to pull up a bank statement and reconcile it with the account in this accounting software.

This might look like the following:
- A user selects the account they want to reconcile
- Then they enter the date and closing balance
- If the totals match, an indication is provided to the user that the reconciliation date is updated
- If the totals don't match, the user is notified and the list of transactions between the last reconciliation date and the date they entered is displayed
